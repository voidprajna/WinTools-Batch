# WinTools-Batch

一个实用的 Windows 系统管理批处理脚本集合，为系统管理员和普通用户提供便捷的系统维护工具。

## 📁 项目简介

本项目收集了多个常用的 Windows 批处理脚本，涵盖系统维护、网络管理、硬件检测等多个方面。所有脚本均为 [.bat](file://d:\Code\demo2603\bat\刷新图标缓存.bat) 格式，可直接在 Windows 系统上运行。

## 🚀 功能特性

### 系统维护
- **刷新图标缓存** - 快速清理和重建系统图标缓存，解决图标显示异常问题
- **修复打印机报错** - 修改打印机驱动程序安装策略，解决打印机相关问题
- **右键样式切换** - 切换 Windows 右键菜单样式（经典/现代）

### 网络管理
- **查找设备 IP** - 快速扫描和发现局域网内的设备 IP 地址
- **刷新域策略** - 强制更新组策略配置
- **刷新所有网络** - 重置网络适配器和网络配置
- **网卡状态切换** - 图形化界面管理网卡启用/禁用状态

### 硬件工具
- **电池报告** - 生成详细的笔记本电池使用报告（支持最近 7 天数据）
- **蓝牙状态切换** - 快速开启或关闭蓝牙功能

### 其他工具
- **重启至 PE** - 一键重启到 Windows PE 环境
- **IE 快捷方式创建** - 创建 Internet Explorer 快捷方式
- **清理脚本** - 系统垃圾清理工具

## 📋 使用说明

### 运行要求
- Windows 7/8/10/11 操作系统
- 部分脚本需要**管理员权限**运行

### 使用方法
1. 克隆或下载本项目到本地
2. 根据需要选择对应的脚本文件
3. 右键点击脚本，选择"以管理员身份运行"
4. 按照屏幕提示操作
